# Crypto Trading Simulator

Web-based Single Page Application (SPA) for learning how to trade cryptocurrencies without financial risk. Built for my Capstone Project.

## Features
* **Real-time Prices:** Fetches live data using the CoinGecko API.
* **Virtual Trading:** Users start with $10,000 to buy and sell crypto.
* **Limit Orders:** Background system that executes trades when target prices are met.
* **Portfolio Analytics:** Calculates Profit/Loss (P/L) and includes interactive charts.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JS
* **Backend:** PHP
* **Database:** MySQL

## How to run locally
1. Clone or download the repository.
2. Place the files in your local server directory (e.g., `htdocs` for XAMPP).
3. Open `phpMyAdmin`, create a database named `cryptosim`, and import the provided `cryptosim.sql` file.
4. Open the project in your browser.
